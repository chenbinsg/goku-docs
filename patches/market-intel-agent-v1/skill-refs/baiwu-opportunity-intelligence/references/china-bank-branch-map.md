# China bank and branch map

This file helps the agent monitor opportunity signals from banks, sub-branches,
rural banks, and consumer-finance-related financial institutions.

## Why this matters for Baiwu

Banks and bank branches often create demand for:

- SMS verification, transaction notification, and service reminders
- customer contact modernization
- AI customer service and knowledge workflow
- outbound service / collection / voice notification
- model governance and routing for regulated AI usage

## Branch-level signals to watch

Do not only monitor head office announcements.

Important branch patterns:

- 总行
- 分行
- 支行
- 信用卡中心
- 远程银行中心
- 运营中心
- 客服中心
- 数据中心 / 科技部 / 金融科技部

Branch-level projects can still matter when they imply:

- regional rollout
- replicable customer-contact workflow
- local AI service deployment
- collections / reminders / service-center transformation

## Output requirement

For bank leads, always try to capture:

- bank name
- branch level
- province / city
- project scope
- whether the signal is replicable to peer branches
- whether the project is compliance-heavy or service-contact-heavy
